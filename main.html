Question-1: What are the different ways to select an element in the DOM?

getElementById(): Selects an element by its unique ID.
getElementsByClassName(): Selects all elements with a specific class name.
getElementsByTagName(): Selects all elements with a specific tag name.
querySelector(): Selects the first element that matches a CSS selector.
querySelectorAll(): Selects all elements that match a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML markup contained within an element, including tags and text.
innerText: Gets or sets the visible text content of an element, ignoring HTML tags and scripts. It is aware of styling and will not return the text of hidden elements.
textContent: Gets or sets the text content of an element and its descendants, including text within script and style tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach an event listener to a parent element instead of attaching it to each individual child element. This is efficient for managing events on many elements, especially when elements are dynamically added or removed.

Question-4: What is event bubbling in the DOM?

Event bubbling is the order in which event handlers are called when an event occurs on a nested element. The event "bubbles" up the DOM tree, starting from the target element where the event occurred and moving up to its parent, grandparent, and so on, until it reaches the root of the document or an event handler stops the propagation.

Question-5: How do you create, add, and remove elements using JavaScript?

Create:
document.createElement('tagname') to create a new HTML element.
Add:
appendChild(): Adds a node as the last child of a parent node.
insertBefore(): Inserts a node before a specified child node.
Remove:
removeChild(): Removes a child node from a parent node.
remove(): Removes the element itself from the DOM.
